# 午前2時の独り言 - 深夜ラジオ番組LP

架空の深夜ラジオ番組「午前2時の独り言」のランディングページです。

## 概要

- **種別**: ランディングページ（lp-）
- **業種**: ラジオ番組・メディア
- **目的**: 番組紹介・過去放送アーカイブ・メッセージ受付
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: ブラック・蛍光グリーン（レトロターミナル風）
- **レイアウト**: 砂嵐ノイズ・グリッチエフェクト
- **ターゲット**: 深夜番組リスナー・レトロ文化愛好家

## ページ構成

1. **ヒーローセクション** - 番組タイトル・キャッチコピー
2. **アーカイブ** - 過去の放送エピソード一覧
3. **メッセージフォーム** - リスナーからのメッセージ受付
4. **フッター** - 番組情報

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts（Share Tech Mono, Zen Old Mincho）

## ファイル構成

```
lp-night-radio/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
└── README.md     # このファイル
```
