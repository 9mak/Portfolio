# 静火 SHIZUBI - 焚き火台ランディングページ

架空の焚き火台ブランド「静火（SHIZUBI）」のランディングページです。

## 概要

- **種別**: ランディングページ（lp-）
- **業種**: アウトドアギア・焚き火台販売
- **目的**: ブランドコンセプトの伝達・製品紹介
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: 黒・オレンジ・炎色のダークトーン
- **レイアウト**: フルスクリーンヒーロー・縦長スクロール
- **ターゲット**: ソロキャンプ愛好家・道具にこだわる層

## ページ構成

1. **ヒーローセクション** - メインビジュアル・ブランド名
2. **コンセプト** - ブランド哲学の説明
3. **製品紹介** - 焚き火台の詳細・スペック
4. **ギャラリー** - 使用イメージ写真
5. **フッター** - コピーライト

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts（Shippori Mincho）
- Canvas API（火の粉パーティクル）

## ファイル構成

```
lp-bonfire-gear/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
├── script.js     # 火の粉エフェクト・フェードイン
└── README.md     # このファイル
```

## 特徴

- レスポンシブ対応
- Canvas APIによる火の粉パーティクルエフェクト
- スクロール連動フェードインアニメーション
- 炎のゆらぎを模したテキストエフェクト
- 赤熱する鉄の表現（CSSアニメーション）
- ダークモードデザイン
