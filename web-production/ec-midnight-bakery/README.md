# 月夜のベーカリー - 深夜営業ベーカリーECサイト

架空の深夜営業ベーカリー「Moonlight Bakery（月夜のベーカリー）」のECサイトです。
深夜の静寂と焼きたてパンの温もりを、ダークブルーの配色と湯気アニメーションで表現しています。

## 概要

- **種別**: ECサイト（ec-）
- **業種**: 深夜営業ベーカリー
- **目的**: オンライン販売・ブランドの世界観訴求
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: 深夜の青 × 月光の黄色（#1a237e、#000022、#fff9c4、#ffd600）
- **レイアウト**: シンメトリックな商品配置・ミニマル
- **ターゲット**: 夜型ライフスタイルの20〜40代

## ページ構成

1. **ナビゲーション** - ロゴ・営業時間
2. **ヘッダー** - 焼き上がりカウントダウン・キャッチコピー
3. **商品一覧** - 3つのパン商品（グリッド表示）

## 技術的な特徴

### CSS アニメーション
- **湯気エフェクト**: `@keyframes`による上昇・拡散アニメーション
- **発光表現**: `drop-shadow`によるパンの温かみ表現
- **ホバーエフェクト**: スケールと発光の強調

### 背景グラデーション
- `radial-gradient`で深夜空を表現

### タイマーボックス
- `backdrop-filter: blur()`によるグラスモーフィズム

## 使用技術

- HTML5
- CSS3（アニメーション、フィルター、グラデーション）
- Tailwind CSS（CDN）
- Google Fonts（DM Serif Display, Zen Kurenaido）
