# Shoji Modal - 障子モーダル

日本の障子を模したモーダルダイアログUI部品です。

## 概要

- **種別**: UI部品（part-）
- **機能**: モーダルダイアログ
- **用途**: 和風サイトのお知らせ表示・情報モーダル
- **制作期間**: 1日

## デザインコンセプト

- **スタイル**: 日本の伝統的な障子戸
- **アニメーション**: 左右に開閉する障子の動き
- **カラースキーム**: 白い障子紙・茶色の木枠

## 機能

1. **障子の開閉** - 左右に分かれて開閉するアニメーション
2. **桟（さん）の表現** - CSSグリッドによる格子模様
3. **影の演出** - 葉影のような装飾
4. **コンテンツ表示** - 障子が閉まってから文字表示

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts（Hina Mincho）
- CSS Transitions（cubic-bezier）

## ファイル構成

```
part-shoji-modal/
├── index.html    # メインファイル
├── style.css     # スタイル
├── script.js     # モーダル開閉制御
└── README.md     # このファイル
```

## 特徴

- 障子の左右開閉アニメーション
- 桟（さん）の格子模様表現
- タイミング制御されたコンテンツ表示
- 和風庭園の背景画像
- SVG影の演出
- トグル機能（開閉）
- レスポンシブ対応
- 和風フォントの使用
