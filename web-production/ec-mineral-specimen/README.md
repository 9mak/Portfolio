# THE SPECIMEN - 鉱物・化石標本ECサイト

架空の鉱物・化石標本専門店「THE SPECIMEN」のECサイトです。

## 概要

- **種別**: ECサイト（ec-）
- **業種**: 鉱物・化石標本販売
- **目的**: 標本コレクターへのオンライン販売
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: ダークブルー・スレートグレー系の標本箱をイメージ
- **レイアウト**: 標本箱を模したグリッドレイアウト
- **ターゲット**: 鉱物・化石コレクター、博物学愛好家

## ページ構成

1. **ヘッダー** - ブランドタイトル
2. **標本グリッド** - 鉱物・化石の一覧表示

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts（Cinzel, Lato）

## ファイル構成

```
ec-mineral-specimen/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
├── script.js     # インタラクション制御
└── README.md     # このファイル
```

## 特徴

- レスポンシブ対応
- ホバー時のルーペ効果
- マウス追従する光沢エフェクト
- 標本箱風のグリッドデザイン
- ラベルプレート付きの標本表示
