# 廃校スタジオ 放課後 - レンタルスタジオサイト

架空の廃校を活用したレンタルスタジオ「放課後」のウェブサイトです。

## 概要

- **種別**: Webサイト（site-）
- **業種**: レンタルスタジオ・撮影スペース
- **目的**: スタジオ紹介・予約案内
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: セピア・ベージュ・ノスタルジックな色調
- **レイアウト**: 懐かしさを感じさせる学校風デザイン
- **ターゲット**: 写真撮影・映像制作関係者

## ページ構成

1. **ヒーローセクション** - メインビジュアル・キャッチコピー
2. **スタジオ紹介** - 施設の特徴・雰囲気
3. **フォトギャラリー** - スタジオ内の写真
4. **料金・予約** - 利用料金・予約フォーム
5. **アクセス** - 所在地・交通案内

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts

## ファイル構成

```
site-closed-school/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
├── script.js     # インタラクション
└── README.md     # このファイル
```

## 特徴

- レスポンシブ対応
- ノスタルジックなデザイン
- 学校の雰囲気を活かしたレイアウト
- 写真ギャラリー
- 予約・問い合わせ機能
