# 【制作実績】中古カメラ買取専門店「光影堂」ランディングページ

架空の中古カメラ買取専門店「光影堂（こうえいどう）」のランディングページです。
信頼感と専門性を感じさせるデザインと、ユーザーが迷わず査定へ進める導線を意識して制作しました。

## 🚀 デモサイト
**[こちらから実際のページをご覧いただけます](https://9mak.github.io/Portfolio/web-production/lp-camera-koei/index.html)**

## 🛠 使用技術
- **Frontend:** HTML5, Tailwind CSS
- **Design:** Google Fonts (Noto Serif JP / Noto Sans JP), Material Symbols
- **Tool:** VS Code, Git/GitHub

## 💡 制作のポイント
1. **ターゲットに合わせたデザイン（トーン＆マナー）**
   - 信頼感と高級感を演出するため、ネイビー (#1a2c4c) と ゴールド (#c5a059) を基調とした配色を採用。
2. **コンバージョンを意識した導線設計**
   - スクロールに追従するCTAボタンを配置し、ユーザーのアクションを促します。
3. **レスポンシブ対応**
   - スマホ・PCの全デバイスで最適に表示される設計です。
