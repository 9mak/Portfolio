# AGED - ビスポーク革靴ECサイト

架空の高級革靴ブランド「AGED（エイジド）」のECサイトです。
経年変化（エイジング）の美しさを体感できるBefore/After比較スライダーを実装し、商品の価値を視覚的に訴求しています。

## 概要

- **種別**: ECサイト（ec-）
- **業種**: 高級革靴・ビスポークシューズ
- **目的**: オンライン販売・ブランディング・経年変化の訴求
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: レザーブラウン × ゴールド（#1a120b、#3e2723、#c5a059）
- **レイアウト**: ダークトーンで高級感を演出
- **ターゲット**: レザーアイテム好き・経年変化を楽しむ30〜50代

## ページ構成

1. **ヘッダー** - ブランドロゴ・ナビゲーション
2. **ヒーローセクション** - メインビジュアル・キャッチコピー
3. **エイジングギャラリー（Before/Afterスライダー）** - 新品と5年後の比較
4. **商品ラインナップ** - 3つの商品カード
5. **フッター** - コピーライト

## 技術的な特徴

### Before/After 比較スライダー
- マウス移動・タッチ操作に連動してスライダーが動く
- `clip-path`を使用した滑らかな画像切り替え
- レスポンシブ対応

### レザーテクスチャ
- SVGフィルターによる微細なノイズ表現
- リアルな革の質感を再現

### 革タグ風ボタン
- `::before`疑似要素で縫い目を表現
- ホバー時のアニメーション

## 使用技術

- HTML5
- CSS3（clip-path、疑似要素、SVGフィルター）
- JavaScript（イベントリスナー、DOM操作）
- Tailwind CSS（CDN + カスタム設定）
- Google Fonts（Cinzel, Noto Serif JP）
