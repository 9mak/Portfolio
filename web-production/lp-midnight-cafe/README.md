# 純喫茶 琥珀 - 深夜カフェランディングページ

架空の深夜営業カフェ「純喫茶 琥珀」のランディングページです。

## 概要

- **種別**: ランディングページ（lp-）
- **業種**: 深夜カフェ・バー
- **目的**: ブランディング・雰囲気の演出
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: ダークネイビー・アンバー・ネオンカラー
- **レイアウト**: ネオンサイン風・雨の演出
- **ターゲット**: 深夜の静かな時間を求める大人層

## ページ構成

1. **ヒーローセクション** - メインメッセージ・雰囲気訴求
2. **メニュー** - 深夜メニュー（ドリンク・軽食）
3. **店舗情報** - 営業時間・アクセス

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts（Hina Mincho, Kaisei Tokumin）
- Canvas API（雨のアニメーション）

## ファイル構成

```
lp-midnight-cafe/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
├── script.js     # 雨のアニメーション
└── README.md     # このファイル
```

## 特徴

- レスポンシブ対応
- Canvas APIによる雨のアニメーション
- ぼかし背景・ネオンサイン風テキスト
- ガラス曇り感の表現
- メニューカードのホバーエフェクト
- ダークモードデザイン
- 深夜の雰囲気を重視したUI
