# KURO COFFEE - スペシャルティコーヒーECサイト

架空のスペシャルティコーヒー専門店「KURO COFFEE」のECサイトです。

## 概要

- **種別**: ECサイト（ec-）
- **業種**: コーヒー豆販売
- **目的**: オンライン販売・ブランディング・定期購入獲得
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: コーヒーブラウン系のアースカラー
- **レイアウト**: 洗練されたミニマルなデザイン
- **ターゲット**: コーヒー好きの20〜40代

## ページ構成

1. **ヒーローセクション** - ブランドメッセージ
2. **特徴バー** - 送料無料・焙煎したて等のアピール
3. **商品一覧** - カテゴリタブ付き商品グリッド
4. **私たちについて** - ブランドストーリー
5. **定期便プラン** - サブスクリプション紹介
6. **ご利用ガイド** - 注文フロー
7. **ニュースレター** - メルマガ登録
8. **フッター** - サイトマップ・会社情報

## 使用技術

- HTML5
- Tailwind CSS（CDN）
- Google Fonts（Playfair Display, Noto Sans JP）
- Material Symbols

## ファイル構成

```
ec-coffee-shop/
├── index.html    # メインページ
├── style.css     # 追加スタイル
├── README.md     # このファイル
└── images/       # 画像（プレースホルダー）
```

## 特徴

- レスポンシブ対応
- EC特有のUI（カートアイコン、商品カード、価格表示）
- サブスクリプションプランの提案
- ブランドストーリーによる信頼感の醸成
- カテゴリフィルター機能（UIのみ）
