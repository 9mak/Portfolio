# Ink Input Component - インク滲み入力欄

万年筆のインクが滲む表現を加えた入力フィールドUI部品です。

## 概要

- **種別**: UI部品（part-）
- **機能**: テキスト入力フィールド
- **用途**: サインフォーム・高級感のある入力UI
- **制作期間**: 1日

## デザインコンセプト

- **スタイル**: 万年筆で書いたようなインク表現
- **カラースキーム**: 生成り紙・ミッドナイトブルーインク
- **エフェクト**: SVGフィルターによるインク滲み

## 機能

1. **紙の罫線** - ノート風の背景デザイン
2. **筆記体フォント** - 手書き感のある入力
3. **インク滲みエフェクト** - SVGフィルター適用
4. **カスタムキャレット** - インク色と同じカーソル

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts（Dancing Script, Shippori Mincho）
- SVG Filters（feTurbulence, feDisplacementMap）

## 特徴

- SVGフィルターによるリアルなインク滲み表現
- 手書き風の筆記体フォント
- 罫線入り背景
- カスタムプレースホルダー
- フォーカス時のボーダー色変化
- レスポンシブ対応
