# 迷宮古書店 積読 - 古書店サイト

架空の古書店「積読（つんどく）」のウェブサイトです。

## 概要

- **種別**: Webサイト（site-）
- **業種**: 古書店・書店
- **目的**: 店舗紹介・在庫検索・オンライン販売
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: アンティークブラウン・ページイエロー・インクブラック
- **レイアウト**: 本に囲まれた空間・迷宮のような雰囲気
- **ターゲット**: 古書愛好家・読書家

## ページ構成

1. **ヒーローセクション** - 店舗の雰囲気を伝えるビジュアル
2. **店舗紹介** - 古書店のこだわり・歴史
3. **蔵書検索** - ジャンル別・著者別検索
4. **注目の一冊** - おすすめ古書の紹介
5. **アクセス** - 店舗情報・営業時間

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts

## ファイル構成

```
site-old-bookstore/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
├── script.js     # 検索・フィルター機能
└── README.md     # このファイル
```

## 特徴

- レスポンシブ対応
- 本棚をイメージしたレイアウト
- アンティーク感のあるデザイン
- 蔵書検索機能
- 読書欲をそそるビジュアル
