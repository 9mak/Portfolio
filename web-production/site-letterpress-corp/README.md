# 活版印刷所 文選 BUNSEN - 活版印刷所サイト

架空の活版印刷所「文選（BUNSEN）」のウェブサイトです。

## 概要

- **種別**: Webサイト（site-）
- **業種**: 活版印刷・印刷業
- **目的**: 工房紹介・作品紹介・受注案内
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: インクブラック・紙ホワイト・活字グレー
- **レイアウト**: クラフト感・手仕事の温もり
- **ターゲット**: デザイナー・こだわりのある印刷物を求める層

## ページ構成

1. **ヒーローセクション** - ブランドメッセージ
2. **工房紹介** - 活版印刷の魅力・こだわり
3. **作品ギャラリー** - 制作実績
4. **サービス** - 名刺・招待状などの印刷物
5. **問い合わせ** - 見積もり・相談フォーム

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN・カスタム設定）
- Google Fonts

## ファイル構成

```
site-letterpress-corp/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
├── script.js     # インタラクション
└── README.md     # このファイル
```

## 特徴

- レスポンシブ対応
- 活版印刷の質感を表現したデザイン
- 手仕事の温もりを感じるレイアウト
- 作品ポートフォリオ
- クラフト感のあるフォント使用
