# 登山ガイド 頂 ITADAKI - 登山ガイドサイト

架空の登山ガイドサービス「頂（ITADAKI）」のウェブサイトです。

## 概要

- **種別**: Webサイト（site-）
- **業種**: 登山ガイド・アウトドアサービス
- **目的**: サービス紹介・ツアー案内・予約受付
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: マウンテングリーン・ストーングレー・スカイブルー
- **レイアウト**: 力強さと自然の雄大さ
- **ターゲット**: 登山愛好家・アウトドア初心者

## ページ構成

1. **ヒーローセクション** - メインビジュアル・キャッチコピー
2. **ガイド紹介** - ガイドのプロフィール・実績
3. **ツアー一覧** - 各種登山ツアーの紹介
4. **装備・準備** - 登山に必要な装備情報
5. **予約・問い合わせ** - ツアー予約フォーム

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts
