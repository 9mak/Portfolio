# Analog Record Player UI - レコードプレイヤーUI部品

アナログレコードプレイヤーを模したインタラクティブUI部品です。

## 概要

- **種別**: UI部品（part-）
- **機能**: レコードプレイヤーの操作シミュレーション
- **用途**: 音楽プレイヤーUI・ローディングアニメーション
- **制作期間**: 1日

## デザインコンセプト

- **スタイル**: アナログレコードプレイヤーのリアルな再現
- **カラースキーム**: ウッドカラー・ブラック・シルバー
- **インタラクション**: トーンアームのクリックで再生/停止

## 機能

1. **レコード盤** - グルーブ（溝）表現・回転アニメーション
2. **トーンアーム** - クリックで位置変更・再生トリガー
3. **ステータスLED** - 再生状態の表示
4. **回転アニメーション** - 再生時のレコード回転

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- CSSアニメーション

## ファイル構成

```
part-analog-record/
├── index.html    # メインファイル
├── style.css     # スタイル
├── script.js     # インタラクション制御
└── README.md     # このファイル
```

## 特徴

- リアルなレコード盤の表現（グルーブ・ラベル）
- トーンアームの回転アニメーション
- クリックでの再生/停止制御
- ステータスLEDの発光表現
- レスポンシブ対応
- トランジション効果
