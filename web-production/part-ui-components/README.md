# UI Components Library - パーツ集

Tailwind CSS ベースのUIコンポーネントライブラリです。

## 概要

- **種別**: パーツ制作（part-）
- **目的**: 再利用可能なUIパーツのショーケース
- **制作期間**: 1日

## 収録コンポーネント

### 1. Buttons
- Primary Buttons（5色）
- Outline Buttons（4種）
- Button Sizes（S/M/L）
- Icon Buttons

### 2. Cards
- Basic Card
- Card with Image
- Stats Card
- User Card

### 3. Forms
- Text Input
- Input with Icon
- Select
- Textarea
- Checkbox / Radio
- Toggle Switch

### 4. Badges
- Solid Badges
- Light Badges
- Pill Badges
- Status Badges with Dot

### 5. Alerts
- Information
- Success
- Warning
- Error

### 6. Navigation
- Tabs
- Pill Tabs
- Breadcrumb
- Pagination

### 7. Avatars
- Various Sizes
- Initials
- With Status Indicator
- Avatar Group

## 使用技術

- HTML5
- Tailwind CSS（CDN）
- Google Fonts（Inter, Noto Sans JP）
- Material Symbols

## ファイル構成

```
part-ui-components/
├── index.html    # コンポーネントギャラリー
├── style.css     # 追加スタイル
├── README.md     # このファイル
└── images/       # 画像（プレースホルダー）
```

## 特徴

- サイドバーナビゲーション付き
- レスポンシブ対応
- コピー＆ペーストで使用可能
- ダークモード対応準備済み
- アクセシビリティを考慮した設計
