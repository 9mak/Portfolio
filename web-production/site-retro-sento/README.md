# 富士見湯 - 昭和レトロ銭湯サイト

架空の昭和レトロ銭湯「富士見湯」のウェブサイトです。

## 概要

- **種別**: Webサイト（site-）
- **業種**: 銭湯・公衆浴場
- **目的**: 銭湯紹介・営業案内・PR
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: タイルブルー・富士山ブルー・レトロオレンジ
- **レイアウト**: 昭和レトロ・タイル絵・銭湯文化
- **ターゲット**: 銭湯ファン・地域住民・観光客

## ページ構成

1. **ヒーローセクション** - 富士山ペンキ絵のビジュアル
2. **銭湯紹介** - 歴史・特徴・こだわり
3. **施設案内** - お風呂の種類・設備
4. **料金・営業時間** - 利用料金・営業案内
5. **アクセス** - 所在地・周辺情報

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts

## ファイル構成

```
site-retro-sento/

├── style.css     # カスタムスタイル
├── script.js     # インタラクション
└── README.md     # このファイル
```

- レスポンシブ対応
