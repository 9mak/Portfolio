# 醸 Kamosu - 自家製味噌キットECサイト

架空の発酵食品専門店「醸（Kamosu）」の味噌キット販売ECサイトです。
スクロール連動で味噌の発酵・熟成過程を視覚的に表現し、商品の価値を体験してもらうことを目的としています。

## 概要

- **種別**: ECサイト（ec-）
- **業種**: 発酵食品・味噌キット販売
- **目的**: オンライン販売・商品の価値訴求・ブランド体験
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: 木目調・味噌色のアースカラー（#d7ccc8、#5d4037、#8d6e63）
- **レイアウト**: スクロールストーリーテリング型
- **ターゲット**: 発酵食品に興味がある20〜50代

## ページ構成

1. **ヘッダー** - ブランドロゴ・タイトル
2. **発酵タイムライン（スクロール連動）** - 仕込み〜完成まで色が変化
3. **商品購入エリア** - 2つの商品プラン（樽付き/袋仕込み）

## 技術的な特徴

### スクロール連動アニメーション
- `IntersectionObserver`や`getBoundingClientRect()`を使用
- スクロール量に応じて背景色・テキスト・画像が段階的に変化
- 味噌の発酵過程を視覚的に表現（仕込み → 3ヶ月 → 6ヶ月 → 10ヶ月 → 1年）

### アニメーション
- CSS `@keyframes`による有機的なブレス（呼吸）アニメーション
- 味噌の塊（blob）を表現する`border-radius`

## 使用技術

- HTML5
- CSS3（カスタムプロパティ、アニメーション）
- JavaScript（スクロールイベント、DOM操作）
- Tailwind CSS（CDN）
- Google Fonts（Noto Serif JP）
