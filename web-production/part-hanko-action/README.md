# Digital Hanko Action - デジタル印鑑アニメーション

和風デジタル印鑑の捺印アニメーションUI部品です。

## 概要

- **種別**: UI部品（part-）
- **機能**: クリックで印鑑捺印アニメーション
- **用途**: 承認フロー・電子署名UI・和風アプリケーション
- **制作期間**: 1日

## デザインコンセプト

- **スタイル**: 日本の印鑑文化をデジタル化
- **カラースキーム**: 朱色・ホワイト
- **アニメーション**: 押印の物理的な動きを再現

## 機能

1. **稟議書UI** - 承認書類のデザイン
2. **承認エリア** - クリック可能な印鑑エリア
3. **捺印アニメーション** - スケール・回転を伴う押印表現
4. **Mix-blend-mode** - 紙に馴染む印影の表現

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts（Hina Mincho, Noto Serif JP）
- CSS Animation（cubic-bezier）

## ファイル構成

```
part-hanko-action/
├── index.html    # メインファイル
├── style.css     # スタイル
├── script.js     # 捺印アクション
└── README.md     # このファイル
```

## 特徴

- リアルな捺印アニメーション
- 沈み込み・回転の表現
- 朱色の印影デザイン
- Mix-blend-modeによる紙への馴染み
- ワンクリックで捺印完了
- 和風フォントの使用
- 状態管理（押印後は無効化）
