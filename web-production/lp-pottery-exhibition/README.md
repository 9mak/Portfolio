# 陶芸家 織部蓮 個展「土と炎」 - 個展ランディングページ

架空の陶芸家「織部蓮」の個展を紹介するランディングページです。

## 概要

- **種別**: ランディングページ（lp-）
- **業種**: アート・工芸展示
- **目的**: 個展の告知・作品紹介・作家プロフィール
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: 和紙色・土色・墨色などの和のカラーパレット
- **レイアウト**: 縦書き・墨の滲み表現
- **ターゲット**: 陶芸愛好家・アートコレクター

## ページ構成

1. **オープニングアニメーション** - ローディング演出
2. **ヒーローセクション** - メインビジュアル・個展タイトル
3. **コンセプト** - 作家の想い・制作哲学
4. **作品** - 展示作品のギャラリー
5. **作家プロフィール** - 経歴・受賞歴
6. **開催概要** - 会期・会場・アクセス情報
7. **フッター** - SNSリンク

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- JavaScript（外部ファイル化）
- Tailwind CSS（CDN・カスタム設定）
- Google Fonts（Noto Serif JP, Shippori Mincho）
- Intersection Observer API（スクロールアニメーション）

## ファイル構成

```
lp-pottery-exhibition/

```

## 特徴

- レスポンシブ対応
- オープニングアニメーション
