# 訳あり野菜直売所「泥付き」 - 有機野菜ECサイト

架空の訳あり野菜直売所「泥付き」のECサイトです。

## 概要

- **種別**: ECサイト（ec-）
- **業種**: 訳あり野菜・有機野菜販売
- **目的**: 規格外野菜のオンライン直販
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: 段ボール・土・自然素材をイメージしたアースカラー
- **レイアウト**: ポラロイド写真風の商品カード配置
- **ターゲット**: オーガニック志向の主婦層、食品ロス削減に関心がある層

## ページ構成

1. **ヘッダー** - ブランド名・キャッチコピー
2. **商品一覧** - 訳あり野菜の商品カード（回転配置）
3. **カートボタン** - 固定フローティングボタン

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- Tailwind CSS（CDN）
- Google Fonts（Yomogi, Zen Maru Gothic）

## ファイル構成

```
ec-organic-vege/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
└── README.md     # このファイル
```

## 特徴

- レスポンシブ対応
- 段ボールテクスチャ風の背景デザイン
- ガムテープ風見出し
- ポラロイド写真風の商品カード
- 手書き風フォントによる親しみやすさ
- 値札シール風の価格表示
- ホバー時の回転・拡大アニメーション
