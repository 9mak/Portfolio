# NexTech Solutions - コーポレートサイト

架空のテック企業「NexTech Solutions」のコーポレートサイトです。

## 概要

- **種別**: コーポレートサイト（site-）
- **業種**: ITソリューション企業
- **目的**: 企業ブランディング・サービス紹介・問い合わせ獲得
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: ブルー × グリーンのグラデーション（テクノロジー感・信頼性）
- **レイアウト**: モダンでクリーンなデザイン
- **ターゲット**: BtoB企業、経営者・IT担当者

## ページ構成

1. **ヒーローセクション** - キャッチコピーとCTA
2. **実績数値** - プロジェクト数・満足度などの統計
3. **会社概要** - 企業理念・特徴
4. **サービス紹介** - 6つのサービスカード
5. **実績紹介** - 4つのプロジェクト事例
6. **CTA** - 問い合わせ誘導
7. **お問い合わせフォーム** - 入力フォーム
8. **フッター** - 会社情報・リンク

## 使用技術

- HTML5
- Tailwind CSS（CDN）
- Google Fonts（Inter, Noto Sans JP）
- Material Symbols

## ファイル構成

```
site-tech-company/
├── index.html    # メインページ
├── style.css     # 追加スタイル
├── README.md     # このファイル
└── images/       # 画像（プレースホルダー）
```

## 特徴

- レスポンシブ対応（モバイル・タブレット・デスクトップ）
- スムーススクロール
- ホバーアニメーション
- グラデーションを活用したモダンなデザイン
- アイコンによる視覚的なサービス表現
