# Cassette Loader - カセットテープローディングUI

カセットテープを模したローディングアニメーションUI部品です。

## 概要

- **種別**: UI部品（part-）
- **機能**: ローディング表示
- **用途**: ページローディング・データ読み込み中の表示
- **制作期間**: 1日

## デザインコンセプト

- **スタイル**: レトロなカセットテープデザイン
- **カラースキーム**: ピンクグラデーション・ブラック・ホワイト
- **アニメーション**: リール回転による読み込み表現

## 機能

1. **カセットボディ** - リアルなカセット形状
2. **ラベルシール** - カスタマイズ可能なラベルエリア
3. **リール窓** - 透明窓を通して見えるリール
4. **回転アニメーション** - リールの連続回転

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- Tailwind CSS（CDN）
- CSSアニメーション
