# 古民家カフェ 木漏れ日 - カフェランディングページ

架空の古民家カフェ「木漏れ日」のランディングページです。

## 概要

- **種別**: ランディングページ（lp-）
- **業種**: カフェ・飲食店
- **目的**: ブランディング・店舗紹介・集客
- **制作期間**: 1日

## デザインコンセプト

- **カラースキーム**: 生成り・焦げ茶・抹茶色などの和のアースカラー
- **レイアウト**: 手書き風・ポラロイド写真風の温かみのあるデザイン
- **ターゲット**: ゆったりとした時間を求める20〜50代

## ページ構成

1. **ヒーローセクション** - メインビジュアル・キャッチコピー
2. **コンセプト** - 店舗の想い・ストーリー
3. **メニュー** - お飲み物・甘味・軽食
4. **アクセス** - 店舗情報・手書き風地図
5. **フッター** - SNSリンク・コピーライト

## 使用技術

- HTML5
- CSS3（外部ファイル化）
- Tailwind CSS（CDN・カスタム設定）
- Google Fonts（Kaisei Decol, Yomogi, Zen Maru Gothic）

## ファイル構成

```
lp-kominka-cafe/
├── index.html    # メインページ
├── style.css     # カスタムスタイル
└── README.md     # このファイル
```

## 特徴

- レスポンシブ対応
- 木漏れ日を表現したアニメーション効果
- ポラロイド写真風の画像レイアウト
- テープで貼った表現（CSSの擬似要素）
- 湯気のアニメーション
- 縦書きテキスト
- ちぎり絵風の境界線
- 手書き風フォントの活用
- 手書き風地図（SVG使用）
