# Portfolio

## ディレクトリ構成と命名規則
このリポジトリは、副業の実績を管理するためのポートフォリオです。

### 1. Web制作・デザイン (web-production/)
- **site-**: 一般的なウェブサイト
- **lp-**: ランディングページ
- **ec-**: ECサイト
- **part-**: バナー・パーツ単位

### 2. システム開発 (system-development/)
- **app-**: アプリケーション
- **tool-**: 業務自動化・スクリプト
- **ai-**: AI・ChatGPT連携
- **api-**: API・バックエンド

### 基本ルール
- すべて小文字、ケバブケース（ハイフン繋ぎ）を使用すること。
