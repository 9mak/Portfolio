# 📝 ai-text-analyzer

AI APIを使用したテキスト分析・インサイトツールです。文章の感情分析、キーワード抽出、可読性評価などを行います。

## ✨ 特徴

- **感情分析**: ポジティブ/ネガティブ/ニュートラル判定
- **キーワード抽出**: 重要な単語・フレーズを自動抽出
- **可読性スコア**: 文章の読みやすさを数値化
- **文字数・語数カウント**: 詳細な統計情報
- **文章トーン判定**: フォーマル/カジュアル/ビジネス等
- **改善提案**: AIによる文章改善アドバイス
- **複数AIプロバイダー対応**: OpenAI、Claude、Gemini等
- **履歴管理**: 過去の分析結果を保存

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. APIキーを設定
3. すぐに使用開始できます

### 基本操作

1. **APIキー設定**
   - 使用するAIプロバイダーを選択
   - APIキーを入力・保存

2. **テキストを入力**
   - 分析したい文章を入力
   - または貼り付け

3. **分析実行**
   - 「分析」ボタンをクリック
   - AIが自動で分析

4. **結果確認**
   - 感情スコア表示
   - キーワード一覧
   - 改善提案を確認

## 🎯 使用例

### ビジネスメール添削
- メール文面を入力
- トーンが適切か確認
- 改善提案を適用

### ブログ記事分析
- 記事全文を入力
- 可読性スコア確認
- キーワード最適化

### カスタマーレビュー分析
- レビュー文を入力
- ポジティブ/ネガティブ判定
- 重要なフィードバックを抽出

## 🛠 技術スタック

- **OpenAI API**: GPT-4による分析
- **Anthropic Claude API**: 高精度な日本語分析
- **Fetch API**: 非同期API通信
- **LocalStorage**: APIキー・履歴保存
- **Vanilla JavaScript**: ES6+

## 📊 分析項目

### 感情分析
- ポジティブスコア (0-100%)
- ネガティブスコア (0-100%)
- ニュートラルスコア (0-100%)

### テキスト統計
- 文字数
- 語数（日本語/英語対応）
- 段落数
- 平均文長

### 可読性
- Flesch Reading Ease (英語)
- 推定読了時間
- 難易度レベル

## 🔑 APIキー取得方法

### OpenAI
1. [OpenAI Platform](https://platform.openai.com/api-keys) にアクセス
2. アカウント作成・ログイン
3. APIキーを生成

### Anthropic Claude
1. [Anthropic Console](https://console.anthropic.com/) にアクセス
2. アカウント作成
3. APIキーを生成

## 📱 対応ブラウザ

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 🔒 プライバシー

- APIキーはLocalStorageに保存（暗号化なし）
- 分析テキストはAI APIに送信されます
- プロバイダーのプライバシーポリシーを確認してください

## ⚙️ カスタマイズ

### デフォルトプロバイダーの設定

`app.js` の設定値を編集：

```javascript
// デフォルトプロバイダー
const defaultProvider = 'openai'; // 'openai', 'claude', 'gemini'
```
