# 🐙 api-github-profile

GitHub REST APIを使用したユーザー統計ビューアーです。リポジトリ一覧、言語統計、コントリビューション情報を可視化します。

## ✨ 特徴

- **ユーザー情報表示**: アバター、プロフィール、統計
- **リポジトリ一覧**: スター数順・更新日順ソート
- **言語統計**: 使用言語の円グラフ表示
- **コントリビューション**: フォロワー・フォロー数
- **検索機能**: ユーザー名で検索
- **シェア機能**: プロフィール画像エクスポート
- **レスポンシブデザイン**: モバイル対応
- **APIキー不要**: 公開APIで取得可能

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. すぐに使用開始できます

### 基本操作

1. **ユーザー検索**
   - GitHubユーザー名を入力
   - 「検索」ボタンをクリック

2. **プロフィール確認**
   - ユーザー情報表示
   - フォロワー・リポジトリ数確認

3. **リポジトリ一覧**
   - 全リポジトリを表示
   - スター数・言語を確認

4. **言語統計**
   - 使用言語の割合を円グラフで表示
   - 得意分野を可視化

## 🛠 技術スタック

- **GitHub REST API**: ユーザー・リポジトリ情報取得
- **Chart.js**: 言語統計グラフ描画
- **Fetch API**: 非同期API通信
- **Canvas API**: プロフィール画像生成
- **Vanilla JavaScript**: ES6+
- **CSS Grid/Flexbox**: レスポンシブレイアウト

## 📊 表示情報

### ユーザー情報
- アバター画像
- 名前・ユーザー名
- Bio（自己紹介）
- 所属・場所
- Webサイト

### 統計情報
- 公開リポジトリ数
- フォロワー数
- フォロー数
- Gist数

### リポジトリ情報
- リポジトリ名
- 説明文
- スター数
- フォーク数
- 主要言語
- 最終更新日

### 言語統計
- 使用言語の割合（円グラフ）
- バイト数ベースの集計
- トップ10言語

## 🔑 API制限

### 認証なし
- 60リクエスト/時間
- 基本情報のみ取得可能

### 認証あり（Personal Access Token）
- 5000リクエスト/時間
- プライベート情報も取得可能

※本アプリは認証なしで動作します

## ⚙️ カスタマイズ

### デフォルトユーザーの設定

`app.js` の設定値を編集：

```javascript
// デフォルトで表示するユーザー
const defaultUser = 'octocat';
```

### 表示リポジトリ数の変更

```javascript
// 表示するリポジトリ数
const maxRepositories = 30;
```

### ソート順の変更

```javascript
// デフォルトソート
const defaultSort = 'stars'; // 'stars', 'updated', 'name'
```

## 📊 API エンドポイント

### ユーザー情報
```
GET https://api.github.com/users/{username}
```

### リポジトリ一覧
```
GET https://api.github.com/users/{username}/repos
```

### 言語統計
```
GET https://api.github.com/repos/{username}/{repo}/languages
```

## 🔧 トラブルシューティング

### ユーザーが見つからない
- ユーザー名のスペルを確認
- 大文字小文字を確認
- ユーザーが存在するか確認

### API制限エラー
- 1時間に60回まで
- しばらく待ってから再試行
- Personal Access Tokenの使用を検討

### データが表示されない
- インターネット接続を確認
- ブラウザのコンソールでエラー確認
