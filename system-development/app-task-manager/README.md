# 📋 app-task-manager

シンプルで直感的なタスク管理アプリケーションです。LocalStorageを使用してデータを永続化し、優先度管理や進捗可視化などの機能を提供します。

## ✨ 特徴

- **直感的なUI**: シンプルで使いやすいインターフェース
- **優先度管理**: タスクを高・中・低の3段階で管理
- **期限設定**: 各タスクに期限を設定可能
- **フィルタ機能**: すべて・未完了・完了済み・優先度別で表示切替
- **統計表示**: 総タスク数、完了数、達成率をリアルタイム表示
- **データ永続化**: LocalStorageで自動保存
- **レスポンシブデザイン**: スマートフォンからPCまで対応

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. すぐに使用開始できます（インストール不要）

### 基本操作

1. **タスク追加**
   - タスク名を入力
   - 優先度を選択（低・中・高）
   - 期限を設定（任意）
   - 「タスクを追加」ボタンをクリック

2. **タスク完了**
   - タスクのチェックボックスをクリック

3. **タスク削除**
   - 「削除」ボタンをクリック

4. **フィルタリング**
   - フィルタボタンをクリックして表示切替

## 🎨 カスタマイズ

`style.css` の `:root` セクションで色をカスタマイズできます：

```css
:root {
    --primary-color: #4f46e5;
    --success-color: #10b981;
    --warning-color: #f59e0b;
    --danger-color: #ef4444;
}
```
