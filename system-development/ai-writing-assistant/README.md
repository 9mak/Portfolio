# ✍️ ai-writing-assistant

OpenAI APIを使用したAI文章作成アシスタントです。ブログ記事、商品説明文、SNS投稿など、様々な種類の文章をAIが自動生成します。

## ✨ 特徴

- **複数のテンプレート**: ブログ、商品説明、SNS、メール、プレスリリース、広告コピー
- **カスタマイズ可能**: トーン（フォーマル/カジュアル等）、文章の長さを調整
- **モデル選択**: GPT-4（高品質）またはGPT-3.5 Turbo（高速・低コスト）
- **履歴管理**: 生成した文章を自動保存（最大20件）
- **クリップボードコピー**: ワンクリックで結果をコピー
- **セキュア**: APIキーはブラウザのLocalStorageに安全に保存

## 🚀 使い方

### 初期設定

1. `index.html` をブラウザで開く
2. [OpenAI Platform](https://platform.openai.com/api-keys) でAPIキーを取得
3. APIキーを入力して「保存」をクリック

### 文章生成

1. **テンプレート選択**
   - 目的に応じたテンプレートを選択

2. **テーマ入力**
   - 文章の内容やキーワードを入力

3. **オプション設定**
   - トーン: フォーマル、カジュアル、フレンドリー、説得的
   - 長さ: 短い（100字）、中程度（300字）、長い（500字以上）
   - モデル: GPT-4 または GPT-3.5 Turbo

4. **生成**
   - 「AI生成」ボタンをクリック
   - 数秒で文章が生成されます

5. **活用**
   - 生成された文章をコピーして使用
   - 必要に応じて編集・調整

## 📝 テンプレート詳細

### 1. ブログ記事
SEOを意識した読みやすい記事を生成
- 適用例: ハウツー記事、解説記事、コラム

### 2. 商品説明文
購買意欲を高める魅力的な説明文
- 適用例: ECサイト、商品ページ

### 3. SNS投稿
エンゲージメントを高める投稿文
- 適用例: Twitter、Instagram、Facebook

### 4. ビジネスメール
丁寧で分かりやすいメール文
- 適用例: 営業メール、問い合わせ、お知らせ

### 5. プレスリリース
メディア向けの正式な発表文
- 適用例: 新商品発表、イベント告知

### 6. 広告コピー
注目を集めるキャッチコピー
- 適用例: バナー広告、LP、チラシ

## 🛠 技術スタック

- **OpenAI API**: GPT-4 / GPT-3.5 Turbo
- **Fetch API**: HTTPリクエスト
- **LocalStorage**: データ永続化
- **Vanilla JavaScript**: ES6+

## 🔒 セキュリティ

### APIキーの保管
- ブラウザのLocalStorageに保存
- サーバーには送信されません
- HTTPS経由でOpenAIと直接通信

### 注意事項
- APIキーは第三者に共有しないでください
- 公開されたウェブサイトでは使用を避けてください
- 本番環境ではバックエンド経由での実装を推奨

## 🎨 カスタマイズ

### 新しいテンプレートの追加

`app.js` の `templates` オブジェクトに追加：

```javascript
this.templates = {
    // 既存のテンプレート...
    custom: {
        name: 'カスタムテンプレート',
        prompt: 'プロンプト内容...'
    }
};
```

### デフォルト設定の変更

```javascript
// デフォルトモデル
const defaultModel = 'gpt-3.5-turbo';

// デフォルトトーン
const defaultTone = 'casual';
```

## 🔧 トラブルシューティング

### エラー: "Invalid API key"
- APIキーが正しいか確認
- `sk-` で始まる正しい形式か確認

### エラー: "Rate limit exceeded"
- 使用量制限を超えています
- しばらく待ってから再試行

### エラー: "Network error"
- インターネット接続を確認
- ブラウザの開発者ツールでエラー詳細を確認
