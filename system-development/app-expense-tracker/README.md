# 💰 app-expense-tracker

家計簿・支出管理アプリです。日々の支出を記録し、カテゴリ別に集計・可視化できます。

## ✨ 特徴

- **支出記録**: 金額、カテゴリ、メモを簡単に記録
- **カテゴリ別集計**: 食費、交通費、娯楽など自動集計
- **グラフ表示**: 月次・年次のチャート可視化
- **予算管理**: カテゴリごとに予算設定＆アラート
- **統計情報**: 合計、平均、最大支出などを表示
- **データ保存**: IndexedDBで永続化
- **エクスポート**: CSV/JSON形式でダウンロード
- **レスポンシブ**: モバイル・タブレット対応

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. すぐに使用開始できます

### 基本操作

1. **支出を追加**
   - 金額、カテゴリ、日付、メモを入力
   - 「追加」ボタンでリストに記録

2. **支出を確認**
   - リスト表示で全ての支出を確認
   - カテゴリでフィルタリング可能
   - 編集・削除も簡単

3. **統計を確認**
   - 月次・年次の集計グラフ
   - カテゴリ別の円グラフ
   - 予算との比較表示

4. **データ管理**
   - CSVエクスポートでバックアップ
   - JSONインポート・エクスポート

## 🎯 使用例

### 月次支出管理
- 毎日の支出を記録
- 月末に自動集計
- 予算オーバーをアラート通知

### カテゴリ別予算管理
- 食費: 30,000円
- 交通費: 10,000円
- 娯楽: 15,000円
- 予算の使用率を可視化

## 📊 データ形式

### エクスポート形式（CSV）
```csv
日付,カテゴリ,金額,メモ
2026-01-12,食費,1500,ランチ
2026-01-12,交通費,300,電車代
```

### JSON形式
```json
[
  {
    "id": 1,
    "date": "2026-01-12",
    "category": "食費",
    "amount": 1500,
    "memo": "ランチ"
  }
]
```

## 🔧 トラブルシューティング

### データが保存されない
- ブラウザのシークレットモードを無効化
- IndexedDBが有効か確認
- ストレージ容量を確認

### グラフが表示されない
- Chart.jsのCDNが読み込めているか確認
- データが登録されているか確認
