# 📰 api-news-aggregator

複数のニュースソースを集約表示するダッシュボードです。カテゴリ別フィルタリング、キーワード検索、ブックマーク機能を搭載しています。

## ✨ 特徴

- **複数ソース集約**: 複数のニュースAPIから情報取得
- **カテゴリ別表示**: テクノロジー、ビジネス、エンタメ等
- **キーワード検索**: タイトル・内容で検索
- **ブックマーク**: 気になる記事を保存
- **既読管理**: 読んだ記事をマーク
- **レスポンシブデザイン**: モバイル対応
- **リアルタイム更新**: 自動リフレッシュ
- **RSS対応**: RSSフィードも取得可能

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. APIキーを設定（オプション）
3. すぐに使用開始できます

### 基本操作

1. **ニュース表示**
   - トップページで最新ニュース表示
   - カテゴリタブで絞り込み

2. **検索**
   - 検索バーでキーワード入力
   - リアルタイム検索

3. **ブックマーク**
   - 気になる記事を保存
   - 後で読むリスト

4. **既読管理**
   - クリックで既読マーク
   - 未読のみ表示も可能

## 🔑 APIキー取得方法

### News API
1. [NewsAPI.org](https://newsapi.org/) にアクセス
2. 無料アカウント作成
3. APIキーを取得
4. アプリに設定

※無料プランは1日100リクエストまで

## ⚙️ カスタマイズ

### デフォルトカテゴリの設定

`app.js` の設定値を編集：

```javascript
// デフォルトカテゴリ
const defaultCategory = 'technology';
```

### 更新間隔の変更

```javascript
// 自動更新間隔（ミリ秒）
const refreshInterval = 300000; // 5分
```

### 表示件数の変更

```javascript
// 1ページあたりの表示件数
const articlesPerPage = 20;
```

## 🔧 トラブルシューティング

### ニュースが表示されない
- APIキーが正しいか確認
- インターネット接続を確認
- APIのレート制限に注意

### 画像が表示されない
- 記事に画像がない場合があります
- URLが正しいか確認

### 検索結果が出ない
- キーワードを変更してみる
- カテゴリを変更してみる
