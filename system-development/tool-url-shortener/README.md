# 🔗 tool-url-shortener

ブラウザ上で動作するURL短縮ツールです。カスタムエイリアスでURLを短縮し、QRコード生成やクリック数カウントが可能です。

## ✨ 特徴

- **URL短縮**: カスタムエイリアスでURL短縮
- **QRコード生成**: 短縮URLのQRコードを自動生成
- **クリック数カウント**: アクセス数を記録・表示
- **タグ管理**: URLをタグで分類・管理
- **メモ機能**: URLの説明・用途を記録
- **検索**: タイトル・タグ・URLで検索
- **エクスポート**: JSON/CSV形式でバックアップ
- **完全ブラウザ処理**: LocalStorageでデータ保存

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. すぐに使用開始できます

### 基本操作

1. **URLを短縮**
   - 元のURLを入力
   - カスタムエイリアス（任意）を設定
   - タグ・メモを追加
   - 「短縮」ボタンをクリック

2. **短縮URLを利用**
   - 生成された短縮URLをコピー
   - QRコードをダウンロード
   - 短縮URLをクリックで元URLへリダイレクト

3. **管理**
   - 一覧表示で全URLを確認
   - クリック数を確認
   - タグでフィルタリング
   - 編集・削除

## 🎯 使用例

### イベント告知
- イベントページのURLを短縮
- カスタムエイリアス: `event2026`
- QRコードをポスターに掲載
- クリック数で効果測定

### SNS投稿
- 長いURLを短縮して投稿
- タグで分類: `twitter`, `campaign`
- クリック数でエンゲージメント測定

### ドキュメント共有
- 社内資料のURLを短縮
- メモで用途を記録
- チームで共有

## ⚙️ カスタマイズ

```javascript

const baseURL = 'https://short.link/';
```

### エイリアス長の制限

```javascript
// エイリアス最小文字数
const minAliasLength = 3;
```

```json
{

  "clicks": 42,
  "tags": ["work", "docs"],
  "memo": "社内資料",
  "created": "2026-01-12T10:30:00.000Z"

```

## 🔧 トラブルシューティング

### データが保存されない
- ブラウザのLocalStorageが有効か確認
- シークレットモードを無効化
- ストレージ容量を確認

### QRコードが表示されない
- QRCode.jsのCDNが読み込めているか確認
- ブラウザのJavaScriptが有効か確認
