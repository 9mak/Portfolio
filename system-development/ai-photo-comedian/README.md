# 😂 ai-photo-comedian

AI Vision APIを使用した「写真で一言」ジェネレーターです。IPPONグランプリのような面白いキャプション・ツッコミをAIが自動生成します。

## ✨ 特徴

- **写真で一言**: 画像から面白いキャプション/ツッコミを生成
- **複数パターン生成**: 5〜10個のバリエーション
- **スタイル選択**: ボケ、ツッコミ、大喜利風、シュール系
- **いいね機能**: 面白さランキング
- **傑作ギャラリー**: 過去の面白い作品を保存
- **SNSシェア**: 画像+テキストを合成してダウンロード
- **複数AIプロバイダー対応**: OpenAI Vision、Claude Vision
- **完全ブラウザ処理**: プライバシー保護

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. APIキーを設定
3. すぐに使用開始できます

### 基本操作

1. **APIキー設定**
   - OpenAI または Claude を選択
   - APIキーを入力・保存

2. **写真をアップロード**
   - ドラッグ&ドロップ または ファイル選択
   - JPG/PNG/WebP対応

3. **スタイル選択**
   - ボケ: シュールで予想外の発想
   - ツッコミ: 写真の矛盾や面白ポイントを指摘
   - 大喜利: お題に対する秀逸な回答
   - フリー: AIの自由な発想

4. **生成**
   - 「一言生成」ボタンをクリック
   - 5〜10個のキャプション候補が表示

5. **評価・保存**
   - 面白いものに「いいね」
   - ギャラリーに保存
   - SNS用に画像ダウンロード

## 🎭 生成スタイル

### ボケ（Boke）
予想外の視点、シュールな発想
- 例: 「犬が散歩させてる人間、めっちゃ疲れてる」

### ツッコミ（Tsukkomi）
矛盾や面白ポイントを指摘
- 例: 「待って、そのポーズ誰が教えたん」

### 大喜利（Ogiri）
お題に対する秀逸な回答
- 例: 「人生で一番高いもの買った時の顔」

### フリー（Free）
AIの自由な発想
- 例: 予測不能な面白コメント

## 🔑 APIキー取得方法

### OpenAI Vision

2. APIキーを生成

## 🔧 トラブルシューティング

### 画像が認識されない
- 対応フォーマットか確認（JPG/PNG/WebP）
- ファイルサイズを確認（推奨: 5MB以下）
- 画像が鮮明か確認

### キャプションが面白くない
- 違うスタイルを試す
- 写真の構図を変える
- 再生成を試す

### API エラー
- APIキーが正しいか確認
- クレジット残高を確認
- レート制限に注意
