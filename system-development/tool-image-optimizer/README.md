# 🖼️ tool-image-optimizer

ブラウザ上で動作する画像最適化ツールです。ドラッグ&ドロップで簡単に画像を圧縮・変換できます。

## ✨ 特徴

- **ブラウザ完結**: サーバー不要、完全クライアントサイド処理
- **WebP変換**: 次世代フォーマットで大幅なサイズ削減
- **画質調整**: 1-100%の範囲で自由に調整可能
- **リサイズ機能**: フルHD、HD、Web標準など複数プリセット
- **メタデータ削除**: EXIF情報を削除してプライバシー保護
- **バッチ処理**: 複数画像を一括処理
- **統計表示**: 削減率をリアルタイム表示
- **ドラッグ&ドロップ**: 直感的な操作性

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. すぐに使用開始できます

### 基本操作

1. **画像をアップロード**
   - ドラッグ&ドロップ、または「ファイルを選択」
   - 複数画像の同時処理が可能

2. **オプション設定**
   - WebP変換: チェックで次世代フォーマットに変換
   - 画質: スライダーで1-100%を調整（推奨: 70-85%）
   - リサイズ: 目的に応じたサイズを選択
   - メタデータ削除: プライバシー保護のため推奨

3. **ダウンロード**
   - 個別にダウンロード、または一括ダウンロード

## 🎯 使用例

### Web用画像の最適化
- 画質: 80%
- WebP変換: ON
- リサイズ: 1920px (フルHD)
- 削減率: 通常70-90%

### サムネイル作成
- 画質: 70%
- WebP変換: ON
- リサイズ: 400px
- 削減率: 通常90-95%

### SNS投稿用
- 画質: 85%
- WebP変換: OFF (互換性重視)
- リサイズ: 1280px
- メタデータ削除: ON

## 🛠 技術スタック

- **HTML5 Canvas API**: 画像処理

- **Blob API**: 最適化画像の生成
- **Drag and Drop API**: ドラッグ&ドロップ
- **Vanilla JavaScript**: ES6+

- メタデータ削除機能でプライバシー保護

## 📊 パフォーマンス

### 処理速度（目安）
- 1枚 (5MB): 約0.5秒
- 10枚 (50MB): 約5秒
- 処理はブラウザの性能に依存

### 削減率（目安）
- JPG → WebP: 25-35%削減
- PNG → WebP: 50-70%削減
- 画質80%設定時の平均削減率: 60-80%

## 🔧 トラブルシューティング

### 画像が処理されない
- 対応フォーマットか確認 (JPG, PNG, GIF, WebP)
- ファイルサイズが大きすぎないか確認
- ブラウザのコンソールでエラーを確認

### ダウンロードできない
- ポップアップブロッカーを確認
- ブラウザの設定でダウンロードを許可
