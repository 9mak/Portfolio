# 🍅 app-pomodoro-timer

PWA（Progressive Web App）対応のポモドーロタイマーです。25分の集中作業と5分の休憩を繰り返し、生産性を高めます。

## ✨ 特徴

- **PWA対応**: オフライン動作、ホーム画面に追加可能
- **3つのモード**: ポモドーロ（25分）、短い休憩（5分）、長い休憩（15分）
- **自動切替**: 4ポモドーロごとに自動で長い休憩に切替
- **進捗表示**: 円形プログレスバーで視覚的に表示
- **統計機能**: 総ポモドーロ数、総集中時間、連続記録を追跡
- **履歴管理**: 日別のポモドーロ数を記録
- **カスタマイズ**: 各モードの時間を自由に設定
- **通知機能**: デスクトップ通知とサウンド通知
- **ダークモード対応**: システム設定に自動追従
- **データ永続化**: LocalStorageで統計と設定を保存

## 🚀 使い方

### インストール（PWA）

1. ブラウザで `index.html` を開く
2. アドレスバーの「インストール」アイコンをクリック
3. アプリとしてインストール完了

または

1. スマートフォンで開く
2. 「ホーム画面に追加」を選択
3. アプリアイコンが追加されます

### 基本操作

1. **モード選択**
   - ポモドーロ: 25分の作業時間
   - 短い休憩: 5分の休憩
   - 長い休憩: 15分の休憩

2. **タイマー操作**
   - 「開始」: タイマー開始
   - 「一時停止」: タイマー一時停止
   - 「リセット」: タイマーをリセット

3. **設定**
   - 各モードの時間をカスタマイズ
   - 自動開始を有効化
   - サウンド・通知のオン/オフ

## 📊 ポモドーロテクニックとは

イタリア人のフランチェスコ・シリロが考案した時間管理術：

1. **25分集中**: タスクに集中して取り組む
2. **5分休憩**: 短い休憩を取る
3. **4回繰り返し**: 上記を4回繰り返す
4. **15-30分休憩**: 長い休憩を取る

### 効果
- 集中力の向上
- 時間管理の改善
- タスクの見える化
- バーンアウト防止

## ⚙️ カスタマイズ

### タイマー時間の変更

デフォルト値を変更する場合、`app.js` の `modes` オブジェクトを編集：

```javascript
this.modes = {
    pomodoro: { duration: 25, label: '作業時間', color: '#ef4444' },
    short: { duration: 5, label: '短い休憩', color: '#10b981' },
    long: { duration: 15, label: '長い休憩', color: '#3b82f6' }
};
```

### 色のカスタマイズ

`style.css` の `:root` セクション：

```css
:root {
    --pomodoro-color: #ef4444;
    --short-break-color: #10b981;
    --long-break-color: #3b82f6;
}
```

### サウンドの変更

`app.js` の `playSound()` メソッドでカスタマイズ可能：

```javascript
playSound() {
    // Web Audio API を使用してカスタムサウンドを生成
    oscillator.frequency.value = 800; // 周波数を変更
}
```

## 📊 統計機能

### 追跡する指標
- **総ポモドーロ数**: 累積完了数
- **総集中時間**: 分単位
- **今日のポモドーロ**: 本日完了数
- **連続記録**: 連続して記録した日数

### データ保存
すべての統計はLocalStorageに保存され、ブラウザを閉じても保持されます。

## 💡 使用のヒント

### 効果的な使い方
1. **明確な目標**: 各ポモドーロで何をするか明確に
2. **中断を避ける**: 25分間は集中
3. **休憩は必須**: 必ず休憩を取る
4. **記録を確認**: 統計で進捗を確認

- 自動開始: オフ（休憩後に準備時間が取れる）
- サウンド通知: オン
- デスクトップ通知: オン

## 🔧 トラブルシューティング

### PWAがインストールできない
- HTTPS環境で動作させる（localhost除く）
- Service Workerが正しく登録されているか確認
- manifest.jsonが正しく配置されているか確認

### 通知が表示されない
- ブラウザの通知許可を確認
- システムの通知設定を確認
- 設定で通知が有効化されているか確認

### データが保存されない
- LocalStorageが有効か確認
- プライベートブラウジングモードを使用していないか確認
- ストレージ容量を確認
